Background

 태양광 발전량은 1년 기준 80배씩 증가하고 있으며, 2002년 이래로 매년 평균 48%의 성장을 하였고, 에너지 기술 분야에서 가장 빠르게 성장하고 있는 분야이다. 또한 한국 에너지공단 신재생에너지센터의 신재생에너지 신규 보급용량 통계자료에 따르면 태양광의 보급 용량은 매 분기 급증하는 추세이며 2018년 7월 기준 889.9MW로 전체 신재생에너지 신규 보급 용량 중 53%라는 가장 큰 비중을 차지하고 있다.
 태양광 발전은 장기적으로 사용하는 데 있어서 모듈 하나하나의 이상 유무나 수명은 발전효율에 막대한 영향을 미치게 된다. 따라서 모듈의 상태를 지속적으로 판단하고 신속하게 복구할 수 있다면 태양광 발전의 효율에 크게 기여될 것으로 예상된다. 이에 따라 본 논문에서는 태양광 모듈의 상태를 실시간으로 확인하고 판단할 수 있으며, 개인사업자가 편리하게 이용 가능한 시스템에 대해 기술한다. 일사량 센서와 태양광모듈로부터 얻는 일사량과의 데이터 비교를 통해 태양광모듈의 상태를 판단하는 방법에 대해 실험하였으며, 추가적으로 푸시 기능을 통하여 상태 판단에 따른 즉각적인 조치가 이루어질 수 있는 시스템을 구축하였다.

DataBase

S.E팀의 Database구조는 대표적으로 사용자의 정보, 패널의 정보 및 각 패널에서 측정된
일사량 센서의 일사량과 전류를 변화시켜 얻은 일사량에 대한 데이터를 저장한다.

  • Date: OCT 2018

API Server

REST란, "웹에 존재하는 모든 자원(이미지, 동영상, DB 자원)에 고유한 URI를 부여해 활용"하는
것으로, 자원을 정의하고 자원에 대한 주소를 지정하는 방법론을 의미한다고 한다.

S.E팀은 JAVA를 이용하여 Rest API를 구축하였으며, HTTP메소드를 통해
클라이언트(라즈베리파이, UI SERVER)와 통신한다

S.E팀의 API-Document 자세히 알아보기

  • Date: OCT 2018

Server Communication

Ajax(Asynchronous JavaScript and XML, 에이잭스)는 비동기적인 웹 애플리케이션의 제작을
위해 아래와 같은 조합을 이용하는 웹 개발 기법이다.

Ajax를 사용하여 API-Server와 UI-Server가 http통신을 하여 데이터를
Insert, Read, Update, Delete 할 수 있게 해준다.

  • Date: OCT 2018

UI-Server

Node.js는 확장성 있는 네트워크 애플리케이션(특히 서버 사이드) 개발에 사용되는 소프트웨어
플랫폼이다. 작성 언어로 자바스크립트를 활용하며 Non-blocking I/O와 단일 스레드 이벤트
루프를 통한 높은 처리 성능을 가지고 있다.

S.E팀은 Node.js 플랫폼으로 javascript언어와 express(웹 어플리케이션을 만들기 위한 틀을 제공하는 라이브러리) 등을 이용하여 UI-Server를 간단하게 구축하였다.

  • Date: OCT 2018

Client-Server

라즈베리 파이(Raspberry Pi)는 영국 잉글랜드의 라즈베리 파이 재단이 학교와 개발도상국에서
기초 컴퓨터 과학의 교육을 증진시키기 위해 개발한 신용카드 크기의 싱글 보드 컴퓨터이다.

S.E팀의 라즈베리파이는 패널의 전류값을 일사량 으로 가공시킨다. 또한 가공시킨 일사량값과 일사량 센서로부터 측정한 일사량 값은 정밀성을 높이기 위해 중앙값 알고리즘을 적용시켜 HTTP통신을 통해 API SERVER와 통신한다.(여기서 전류 값을 얻어오기 위해 라즈베리파이는 태양광 패널이 연결된 컨트롤러와 시리얼 통신을 하며, 일사량 값은 아두이노와의 시리얼 통신을 통해 가져온다.)

  • Date: OCT 2018

Device

아두이노(Arduino)는 오픈 소스를 기반으로 한 단일 보드 마이크로컨트롤러로 완성된 보드(상품)와 관련 개발 도구 및 환경을 말한다.

S.E팀은 아두이노에 전류센서, 일사량센서, 온도센서, LCD패널을 디바이스에 부착하여 태양광 패널의 이상유무를 확인할 수 있는 휴대용 디바이스를 제작하였다.
